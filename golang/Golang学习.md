# Golang 安装


## 环境配置

设置环境变量
```
GOPATH=D:\develop\code\gopath
GOROOT=D:\develop\app\go
PATH=%GOROOT%\bin


```

设置代理
```
$ go env -w GO111MODULE=on
$ go env -w GOPROXY=https://goproxy.cn,direct
```

关闭代理校验
```
GOSUMDB=off
```
