# Example Project

## 项目简介
Python示例项目，演示常用库（dateutil/PyYAML/requests）的基础用法

## 环境要求
- Python 3.6+
- 依赖库：
  - python-dateutil >=2.5.3
  - PyYAML >=3.11
  - requests >=2.10.0

## 快速开始
```bash
# 安装依赖
pip install -r requirements.txt

# 运行示例代码
python main.py
```

## 功能特性
- 日期时间处理示例
- YAML配置文件解析
- HTTP请求演示
